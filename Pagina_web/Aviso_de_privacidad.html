Aviso de privacidad

Este mensaje tiene el objetivo de informar sobre cómo serán utliizados, recopliados y protegidos los datos personales que puedan ser proporcionados durante el uso de esta aplicación. En caso de tener alguna duda se recomienda leer detenidamente el aviso completo.

  • Uso de datos personales.

    Todos los datos personales que lleguemos a recopliar durante las operaciones internas de esta aplicación tienen los siguientes propósitos:
    • Registrar los datos para el servicio de la aplicación
    • Registrar las transacciones y actividades que se ejecutaran al momento de utliizar la aplicación.
    • Generar un acceso a las funciones y servidores de la plataforma.

  • Datos generales recaudados.

    Los siguientes datos enlistados serán recaudados con el fin de agliizar los procesos y generar estadísticas con el fin de poder ayudar a generar una mejor experiencia dentro de los comedores.

    • Nombre y apellidos.
    • CURP (Clavé Única de Registro de Población).
    • Año de nacimiento.
    • Sexo.
    • Situación vulnerable.

  • Protección de datos personales.

    Nos comprometemos a mantener la confidencialidad de los datos personales que se solicitan registrar durante todo el proceso. Todos los datos registrados se resguardan mediante el uso de distintas técnicas de seguridad, como la codificación de datos, limitación de acceso, entre otras. Con el fin de evitar la divulgación, alteración, destrucción y acceso no autorizado a personas ajenas al proyecto.

  • Compartir datos personales.

    El único fin de compartir los datos personales será única y exclusivamente para cumplir con nuestras obligaciones legales o al momento en el que la persona que haya brindado los datos brinde su consentimiento para ello.
